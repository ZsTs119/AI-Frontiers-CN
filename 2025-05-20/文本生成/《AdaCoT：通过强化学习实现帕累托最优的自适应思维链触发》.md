# AdaCoT - AdaCoT: Pareto-Optimal Adaptive Chain-of-Thought Triggering via Reinforcement Learning

**模型功能**：提出AdaCoT算法，利用强化学习自动调整思维链触发时机，以实现帕累托最优性能，可辅助进行文本相关的逻辑推理和生成等。

**arXiv 文章链接**：
[https://arxiv.org/abs/2505.11896](https://arxiv.org/abs/2505.11896)

**研究进展**：该模型的创新点在于引入强化学习来自动调整思维链触发时机，实现帕累托最优性能，这为文本生成中逻辑推理过程的优化提供了新的思路和方法，区别于传统的思维链触发方式。

**应用场景**：可应用于自动写作、文章续写等文本生成场景，帮助生成更具逻辑和连贯性的文本内容；也可用于聊天机器人等，提升其回答的逻辑性和准确性。
