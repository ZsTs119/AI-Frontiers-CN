# Chain-of-Model Learning - 《Chain-of-Model Learning for Language Model》

**模型功能**：该模型提出了链式模型学习方法，先训练小模型，再逐步融合大模型，以此提升语言模型的性能和泛化能力，能够学习文本的特征和模式来生成新的文本。

**研究进展**：创新点在于提出了链式模型学习方法，打破了传统训练模式，通过先训练小模型再融合大模型的方式，有望提升语言模型在文本生成任务中的性能和泛化能力，进一步推动文本生成领域的发展。

**应用场景**：适用于自动写作、自动翻译、聊天机器人、文章续写、新闻撰写、剧本生成等需要文本生成的场景。
